## 프로젝트 소개
```
로봇의 발전과 이에 따른 시장 구모가 확대가 되어있는 상황에서, 
다양한 분야로의 활용을 경험해보고자 도서관에 접목시켜 
책 정리를 위한 자율주행카트 개발을 실시함
```
